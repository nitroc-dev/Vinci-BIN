# Devops-05-tdd-final

## Description

Node JS API

### Prerequisites

-   [Node JS / NPM](https://nodejs.org/en/)
-   [Yarn (optional)](https://yarnpkg.com/)

### Installing

Install dependencies

```
npm i
```

or

```
yarn
```

## Getting Started

Run the app

```
npm start
```

or

```
yarn start
```

## API Documentation

Import doc/devops-05.postman_collection.json to have the endpoints definition in Postman
